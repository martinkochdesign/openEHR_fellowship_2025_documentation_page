Visualization Tool - Visualization Language

For the visualization tool, a network representation of connections between different archetypes will be visualized. The views defined so far are:

  • Archetype Connection: For a selected archetype, the relationships to other archetypes should be shown.
    • Parent
    • Children
    • Inclusion
    • Exclusion
  • Similar Archetypes: For a selected archetype, similar archetypes should be shown.
  • Collection: For the collection, all archetypes from the collection and their connections should be shown.
  • Project: For the project, the sections and elements of the checklist should be shown with the corresponding assigned archetypes.

For these four views, a clear and intuitive visual representation has to be defined. In the following sections, a proposal for a representation of every view will be elaborated.


Let's get visual.

Archetype Connection

In the Archetype view, all relationships between the active archetype and other existing archetypes are visualized. Different types of connections are represented with distinct symbols and line styles for clarity:

  • Specialized archetypes: Archetypes that are specialized from the active one are shown with a small black disc, an arrow pointing toward the parent, and a solid black line.
  • Parent archetypes: Archetypes that are parents of the active one are indicated by a large black disc, an arrow pointing toward the parent, and a solid black line.
  • Included archetypes: Archetypes referenced in inclusion statements within the slots of the active archetype are marked with a green circle and a green dashed connection line.
  • Excluded archetypes: Archetypes referenced in exclusion statements within the slots of the active archetype are marked with a red circle and a red dashed connection line.

Similar Archetypes

In the Similar view, a collection of archetypes that are similar or related to the active archetype is displayed. These similar archetypes are highlighted in a coral color for easy identification. Notably, there are no direct connections shown between the active archetype and the similar ones in this view.


Legend: Similar archetypes are shown in coral red.

Project

A project is visualized using three columns, arranged from left to right: Checklist Sections, Checklist Elements, and Assigned Archetypes.

Checklist elements are connected to their corresponding sections with a dashed line. When an archetype is assigned to an element, it is displayed in the third column and linked to the element with a dashed line as well.

Color coding of the linked elements:

  • Pink: Section name
  • Green: Element name
  • Yellow: Existing archetype
  • Blue: New archetype
