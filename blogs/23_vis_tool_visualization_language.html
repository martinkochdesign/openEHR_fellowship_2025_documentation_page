Visualization Tool - Visualization Language

For the visualization tool, a network representation of connections between different archetypes will be visualized. The views defined so far are:

  • Archetype Connection: For a selected archetype, the relationships to other archetypes should be shown.
    • Parent
    • Children
    • Inclusion
    • Exclusion
  • Similar Archetypes: For a selected archetype, similar archetypes should be shown.
  • Collection: For the collection, all archetypes from the collection and their connections should be shown.
  • Project: For the project, the sections and elements of the checklist should be shown with the corresponding assigned archetypes.

For these four views, a clear and intuitive visual representation has to be defined. In the following sections, a proposal for a representation of every view will be elaborated.
