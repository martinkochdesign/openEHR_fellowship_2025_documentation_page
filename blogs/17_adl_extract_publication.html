Source Code

The complete source code for the ADL extraction component can be found in the src folder of the repository. Please note that the version number is included in the folder or file names (for example, V4_11 stands for version 4.11).

Executable Release

For users who prefer not to run the source code directly, a compiled executable version of the script is provided in the dist folder . This executable is designed to work on any modern Windows computer, making it easy to get started without additional setup.

Handling large files in GitHub

Because the executable file is bigger than 100MB, the file cannot be oushed directly to the GitHub repository. For larger files, Git Large File Storage (or Git LFS) has to be employed.

For this, the current version at the time (v3.7.0) from Git LFS was installed on the computer with the GitHub Desktop installation.

In the .gitattributes file of the repository a line was added to manage the large .exe files


        # Auto detect text files and perform LF normalization
        * text=auto
        # Manage large exe files eith Git LFS
        *.exe filter=lfs diff=lfs merge=lfs -text
