ADL extraction - Data Source

One of the main functionalities of the application is the possibility to download archetype collection in a ZIP file format from the CKMs or from GitHub repositories.

To get direct access to these zip files, the corresponding URLs have to be used.

Please note that the GitHub mirrors of CKMs exclusively contain archetypes from projects, excluding those from incubators. Conversely, the URLs for the CKM lead to zip files that encompass all archetypes from both incubators and projects.

If there is a need to download CKM data without incubators, the URLs should follow this format:

https://ckm.openehr.org/ckm/retrieveResources?resourceType=archetype&projectType=project&format=ADL&state1=INITIAL&state2=DRAFT&state3=TEAMREVIEW&state4=REVIEWSUSPENDED&state5=PUBLISHED&state6=REASSESS_DRAFT&state7=REASSESS_TEAMREVIEW&state8=REASSESS_REVIEWSUSPENDED
