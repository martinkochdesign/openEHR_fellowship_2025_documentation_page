Visualization tool - GUI

Wireframe/Early draft

The Main View provides access to essential Project Options at the top of the interface. On the right-hand side, you will find the Checklist Sidebar. On the left, the Archetype List Sidebar and Collection view are located. A simple filter in the Archetype List Sidebar serves as the Search function. The Visualization area is situated in the middle section, offering different views to select from.

First Iteration of the GUI

Based on the previously defined wireframe, a first iteration of the GUI was implemented. The Main View closely resembles the initial wireframe.


First GUI developed - Main View layout.

A separate editing view for the checklist was implemented, as shown in the following figure.


First GUI - Checklist Editor view.

This implementation used a fixed grid layout for the views, without allowing users to resize or reposition them. When integrating results from the User Journey Mapping, this rigid grid approach proved problematic, especially when adding additional views to the project.

Second Iteration of the GUI

Using the comprehensive list of "Design Requirements for Different Views/Pages/Windows" from the User Journey Mapping (see details), the second iteration of the GUI was drafted.
